# Bird Detector Scripts

The birdcompare-live.py script is used to test files in the Timelapse folder and move any containing birds into the today folder and deletes the empty ones.

The birdcompare.py script was used to test the accuracy of the threshold levels on a sample of files in the birds and no-birds folders. It then also uses any samples in the Timelapse folder to classify and move images of birds into a today folder
